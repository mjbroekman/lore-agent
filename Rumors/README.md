# Rumors, Myths, and Legends

Use this directory to store rumors, myths, and legends.  Keep each one in a separate, atomic file within a 'data' directory.

## Format
```
---
category: rumor
accuracy: false|true|partial
status: active|historical
tier: common|uncommon|rare|legendary
audience: commoner|classes|leaders
locations:
  - location type 1
  - location type 2
factions:
  - faction 1
  - faction 2
---
# reference name
<description>
```
