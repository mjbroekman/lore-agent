# Description
The Zhiedphisian Pantheon is a matriarchal structure with [Ai Xiang] as the head deity.

# Structure
The Zhiedphisian Pantheon is divided into tiers, based on how separated they are from [Ai Xiang]. If a deity was created directly by or from [Ai Xiang], they are a First Tier deity. Those created by First Tier deities are Second Tier, and so on. There is no difference in power or importance between tiers.
